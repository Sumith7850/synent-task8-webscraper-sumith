# Web Scraper Project

## Description
This project extracts book data from a website and stores it in a structured CSV format.

---

## Features
- Scrapes 200+ records
- Extracts:
  - Title
  - Price
  - Availability
  - Rating (converted to numeric)
- Handles multiple pages
- Cleans and formats data

---

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas

---

## How to Run

1. Clone repository:
git clone https://github.com/your-username/synent-task8-webscraper-sumith.git

2. Install dependencies:
pip install -r requirements.txt

3. Run:
python scraper.py

---

## Output
- books_data.csv with structured data
